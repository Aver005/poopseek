# Tool Parsing Bug

## Проблема
При возврате DeepSeek большого ответа (>2000 токенов) парсер инструментов ломается:
- Инструмент не детектится
- JSON может быть разорван на части

## Коренная причина
`src/agent/tool-call-parser.ts` ожидает полный текст за раз, но ответ приходит частями через HTTP chunking. JSON-блок может быть разорван между чанками.

## Решение
1. Внедрить потоковый парсер с буферизацией в `collectDeepseekOutput`
2. Накопление JSON-блоков до полного закрытия скобок
3. Отдавать завершённые блоки на парсинг по мере поступления
