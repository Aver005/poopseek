# Сборка исполняемого файла

## Важное ограничение Bun

**Bun не поддерживает кросс-компиляцию.** `bun build --compile` создаёт
бинарник под ту ОС, на которой запущен. Нельзя собрать macOS-бинарь
из Linux, и наоборот.

## Варианты сборки

### 1. Docker (Linux-бинарь)

Работает на любой платформе с Docker.
