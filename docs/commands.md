# Команды пользователя

## Обзор

PoopSeek поддерживает slash-команды для управления сессией, моделью и настройками. Команды вводятся в начале строки и обрабатываются до отправки в LLM.

## Список команд

### `/help`
Показывает справку по всем доступным командам.

```
/help
```

### `/tools`
Выводит список доступных инструментов с описаниями.

```
/tools
```

### `/model [name]`
Просмотр или смена модели.

```bash
/model                    # показать текущую модель
/model deepseek-coder     # переключиться на deepseek-coder
```

### `/compact`
Сжимает историю диалога через LLM. Полезно при приближении к лимиту контекста.

```bash
/compact                  # сжать всю историю
/compact 20               # сжать последние 20 сообщений
```

### `/clear`
Очищает историю диалога (без подтверждения).

```
/clear
```

### `/theme [name]`
Управление цветовой темой.

```bash
/theme                    # показать текущую тему
/theme dark               # тёмная тема (по умолчанию)
/theme light              # светлая тема
```

### `/btw <question>`
Sidechat — фоновый запрос без прерывания основного диалога.

```bash
/btw как работает функция map в Python?
```

Ответ будет выведен отдельным блоком, а основной диалог продолжится.

### `/exit` или `/quit`
Завершает работу программы.

```
/exit
```

## Синтаксис и особенности

### Многострочные команды
Некоторые команды (например, `/btw` с длинным вопросом) могут быть продолжены на следующей строке через Shift+Enter.

### Аргументы с пробелами
Если аргумент содержит пробелы, можно использовать кавычки:

```
/model "my-custom-model"
```

### Автокомплит команд
При вводе `/` терминал предлагает доступные команды (через Tab).

## Добавление новой команды

См. `development-guide.md` → раздел "Добавление slash-команды".

## Пример сессии

```
> /model
tекущая модель: deepseek-chat

> /model deepseek-coder
Модель изменена на deepseek-coder

> Напиши функцию на Python для сортировки списка
[ответ модели]

> /btw а как это работает на Go?
[фоновый ответ про Go]

> /compact
Сжатие истории... Готово (15 сообщений → 3)

> /exit
До свидания!
```

## Поведение при ошибках

- **Неизвестная команда**: выводит "Unknown command. Type /help for available commands."
- **Неверные аргументы**: выводит usage для конкретной команды
- **Команда во время генерации**: ставится в очередь, выполнится после завершения текущего ответа
