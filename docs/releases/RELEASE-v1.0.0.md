# PoopSeek v1.0.0 — CLI AI-агент с доступом к файловой системе и shell

**Первый стабильный релиз** интерактивного агента, работающего через DeepSeek API. PoopSeek самостоятельно планирует цепочки действий и использует 8 встроенных инструментов для выполнения задач пользователя.

## ✨ Ключевые возможности

- **Agent Loop** — итеративное планирование и выполнение действий (до 12 циклов на запрос)
- **8 инструментов**: `bash`, `powershell`, `file.read`, `file.write`, `file.edit`, `file.list`, `file.remove`, `tools.list`
- **Цветной CLI с автодополнением**: поддержка тем (dark/light), подсказки по Tab
- **Управление контекстом**: хранение до 40 сообщений, внешние промпты, переменные
- **DeepSeek API с function calling**
- **Высокая производительность** на Bun + TypeScript

## 🛠️ Команды CLI (14 штук)

| Команда | Описание |
|---------|----------|
| `/help` | Показать справку по всем командам |
| `/clear` | Очистить историю диалога |
| `/session` | Показать ID текущей сессии |
| `/stats` | Статистика контекста (количество сообщений, токенов) |
| `/history` | Показать историю сообщений |
| `/theme [dark\|light]` | Сменить цветовую тему |
| `/model [model_name]` | Выбрать модель DeepSeek |
| `/tools` | Список доступных агенту инструментов |
| `/compact` | Сжать контекст (обобщить историю) |
| `/cmpt` | Алиас для `/compact` |
| `/switch [session_id]` | Переключиться на другую сессию |
| `/btw` | Показать "between"-контекст (связи между действиями) |
| `/exit` и `/quit` | Завершить работу |

**Особенности интерфейса**:
- Автодополнение команд по `Tab`
- Подсказки доступных команд во время ввода
- Отображение прогресса выполнения инструментов (иконка 🔨, статус)

## 🤖 Доступные агенту инструменты

| Инструмент | Назначение |
|------------|------------|
| `bash` | Выполнение bash-команд (Linux/macOS/WSL) |
| `powershell` | Выполнение PowerShell команд (Windows) |
| `file.read` | Чтение файла |
| `file.write` | Создание/перезапись файла |
| `file.edit` | Частичное редактирование (замена строк) |
| `file.list` | Список файлов в директории |
| `file.remove` | Удаление файла или папки |
| `tools.list` | Просмотр доступных инструментов |

Агент сам выбирает нужные инструменты и аргументы на основе запроса.

## ⚙️ Архитектурные особенности

- **Гибкие промпты** из внешних файлов (`assets/prompts/base.prompt.md`)
- **Переменные в промптах**: `{{workspaceRoot}}`, `{{dateTime}}` и др.
- **Валидация аргументов** по JSON Schema перед вызовом инструментов
- **Изоляция операций** в рабочей директории
- **Управление сессиями** — поддержка переключения между разными диалогами (`/switch`)
- **Функция Compact** — обобщение длинной истории для экономии токенов

## 🚀 Быстрый старт

**Требования**: Bun v1.0+, DeepSeek token (получается на chat.deepseek.com через `localStorage.getItem('userToken')`)

```bash
git clone <repo-url>
cd poopseek
bun install
echo "DEEPSEEK_TOKEN=ваш_ключ" > .env
bun start
```

## 📦 Сборка исполняемого файла

```bash
bun run build:exe --version 1.0.0
```

При сборке можно указать версию через аргумент `--version`. Если не указана, используется `POOPSEEK_VERSION` → `APP_VERSION` → `dev`.

## 🔧 Технологический стек

- **Runtime**: [Bun](https://bun.sh)
- **Язык**: TypeScript 5+
- **LLM API**: DeepSeek (совместим с OpenAI API)
- **CLI**: нативный Node.js readline + кастомные подсказки и автодополнение
- **Зависимости**: terminal-kit

## ⚠️ Известные ограничения

- Требуется активное интернет-соединение для API DeepSeek
- На Windows bash-инструмент работает только через WSL или Git Bash
- Максимальный размер контекста — 40 сообщений (настраивается)
